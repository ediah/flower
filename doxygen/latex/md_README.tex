{\bfseries{Flower}} — строго типизированный язык процедурного программирования. Языком поддерживается многопоточность, указатели заменены на концепт \char`\"{}общей\char`\"{} (или \char`\"{}разделяемой\char`\"{}) переменной, есть 4 простых типа и один составной. Продвинутая работа со структурами\+: над ними можно проводить арифметические операции.

В репозитории ведётся разработка\+:


\begin{DoxyEnumerate}
\item Оптимизирующего байт-\/код компилятора ({\bfseries{flc}}),
\item Виртуальной стековой машины ({\bfseries{flvm}}),
\item Отладчика виртуальной машины ({\bfseries{fldbg}}).
\end{DoxyEnumerate}\hypertarget{md_README_autotoc_md1}{}\doxysection{Порядок работы}\label{md_README_autotoc_md1}

\begin{DoxyEnumerate}
\item Скомпилировать написанную программу {\ttfamily source.\+fl}\+:

\end{DoxyCode}

\item Запустить бинарный файл на виртуальной машине\+: 
\begin{DoxyCode}{0}

\end{DoxyCode}

\item В случае ошибок или иных проблем отладить\+: 
\begin{DoxyCode}{0}

\end{DoxyCode}

\item Если вносятся изменения в код проекта, все изменения можно автоматически протестировать с помощью команды\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{./fltest [-\/O] [-\/m] [-\/r] ./test/A-\/unit/*.fl ./test/B-\/unit/*.fl}

\end{DoxyCode}
